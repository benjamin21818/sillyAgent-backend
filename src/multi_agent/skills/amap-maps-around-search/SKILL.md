# 周边搜索

根据关键词与坐标搜索半径范围内的 POI。

## MCP 工具名称

maps_around_search

## 关键词

- 周边
- 附近
- 周边搜索
- 附近搜索
- POI
- 餐饮
- 景点
- 医院
- 加油站
- 便利店

## 触发时机

- 搜索附近门店/景点/服务
- 查询指定位置周边 POI

## 使用示例

```json
{ "keywords": "咖啡", "location": "116.481028,39.989643", "radius": "1000" }
```
