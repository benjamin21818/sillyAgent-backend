# 距离测量

测量经纬度坐标之间的距离，支持驾车/步行/直线距离。

## MCP 工具名称

maps_distance

## 关键词

- 距离
- 测距
- 两点距离
- 路程
- 里程
- 直线距离
- 驾车距离
- 步行距离

## 触发时机

- 测量两点距离
- 估算通勤距离

## 使用示例

```json
{ "origins": "116.481028,39.989643", "destination": "116.434446,39.90816", "type": "1" }
```
