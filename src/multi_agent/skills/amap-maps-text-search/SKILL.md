# 关键词搜索

根据关键词进行 POI 搜索并返回结果。

## MCP 工具名称

maps_text_search

## 关键词

- 关键词搜索
- 地点搜索
- POI搜索
- 地址搜索
- 查询地点
- 搜索商户
- 搜索景点

## 触发时机

- 关键词搜索地点
- 查询指定城市内 POI

## 使用示例

```json
{ "keywords": "博物馆", "city": "北京", "citylimit": false }
```
