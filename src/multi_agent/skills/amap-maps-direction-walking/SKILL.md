# 步行路线规划

根据起终点经纬度规划步行通勤方案。

## MCP 工具名称

maps_direction_walking

## 关键词

- 步行
- 走路
- 徒步
- 导航
- 路线

- 路径
- 步行路线

## 触发时机

- 规划步行路线
- 查询步行通勤方案

## 使用示例

```json
{ "origin": "116.481028,39.989643", "destination": "116.434446,39.90816" }
```
