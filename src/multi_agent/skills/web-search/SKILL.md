# 网页搜索

该技能用于通过 MCP 工具 `search_web` 执行网页搜索，并返回结果列表。适用于需要外部资料或实时信息的场景。

## MCP 工具名称

search_web

## 关键词

- 搜索
- 网页
- 联网
- 新闻
- 资料
- 网址
- 检索
- 资料查询
- 最新
- 官网
- 查找

## 触发时机

- 用户需要联网检索某个主题
- 用户请求网页摘要或来源链接
- 需要补充最新资料（新闻、官网文档、发布信息）

## 工具说明

工具名：`search_web`  
参数：
- `query`：搜索关键词
- `max_results`：最大返回结果数（默认 3）

## 使用示例

```json
{ "query": "飞书 文档 API", "max_results": 3 }
```

```json
{ "query": "LangChain MCP 使用方法", "max_results": 5 }
```

## 输出说明

返回列表通常包含标题、链接与摘要字段，可直接用于后续阅读或整理。
