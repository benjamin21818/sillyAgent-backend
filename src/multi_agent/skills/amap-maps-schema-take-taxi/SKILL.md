# 打车唤醒链接

根据起终点信息生成高德打车唤醒链接。

## MCP 工具名称

maps_schema_take_taxi

## 关键词

- 打车
- 叫车
- 出租车
- 网约车
- 代驾
- 高德打车
- 打车链接

## 触发时机

- 生成打车唤醒链接
- 需要唤起高德打车

## 使用示例

```json
{ "slon": "116.481028", "slat": "39.989643", "sname": "起点", "dlon": "116.434446", "dlat": "39.90816", "dname": "终点" }
```
